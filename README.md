# Devis Pro - Application de Gestion de Devis

Application web moderne de gestion de devis avec système d'authentification complet.

## Fonctionnalités

- ✅ Authentification complète (inscription, connexion, vérification email, reset password)
- ✅ Gestion CRUD des devis
- ✅ Lignes de devis dynamiques avec calcul automatique
- ✅ Tableau de bord avec statistiques
- ✅ Design moderne et responsive
- ✅ Sécurité renforcée (CSRF, prepared statements, sessions sécurisées)

## Technologies

- PHP 8.2 + Apache
- PostgreSQL 15
- Docker & Docker Compose
- CSS3 (Grid, Flexbox, variables)
- JavaScript Vanilla (ES6+)

## Installation locale

```bash
# Cloner le projet
git clone [URL]

# Lancer avec Docker
docker-compose up -d

# Initialiser la base de données
# Accéder à http://localhost:8080/init-db.php

# Accéder à l'application
# http://localhost:8080
```

## Déploiement

Déployable sur Railway, Render, ou tout hébergeur supportant Docker.
